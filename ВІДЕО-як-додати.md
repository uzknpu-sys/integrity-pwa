# Як додати відео до довідника

Відео у довіднику працюють як **онлайн-посилання** (YouTube): у застосунку
зберігається лише посилання, саме відео відкривається за наявності інтернету.
Це правильно — відеофайли надто важкі, щоб тримати їх офлайн.

## Куди додавати

Відкрийте файл `guide.html` текстовим редактором і знайдіть фрагмент
`"videos":[` (можна через Ctrl+F). Там зараз один порожній шаблон:

```
"videos":[{"id":"v1","title":"Додайте посилання...","url":"","topic":"...","description":"..."}]
```

## Як додати своє відео

Замініть шаблон на свої записи. Кожне відео — це блок у фігурних дужках:

```
"videos":[
  {"id":"v1","title":"Конфлікт інтересів: як діяти","url":"https://www.youtube.com/watch?v=ВАШ_КОД","topic":"Конфлікт інтересів","description":"Коротке відеороз’яснення на прикладі ситуації."},
  {"id":"v2","title":"Обмеження щодо подарунків","url":"https://www.youtube.com/watch?v=ІНШИЙ_КОД","topic":"Подарунки","description":"Що можна, а що заборонено приймати."}
]
```

Правила:
- кожен блок відокремлюється комою, після останнього коми немає;
- `url` — повне посилання на YouTube;
- `title` — назва, `topic` — тема, `description` — короткий опис;
- лапки `"..."` не прибирайте, інакше довідник не відкриється.

## Після редагування

Завантажте оновлений `guide.html` на GitHub (Add file → Upload files →
перезаписати) і підніміть версію кешу: у файлі `sw.js` змініть
`integrity-v2` на `integrity-v3`, щоб телефони підхопили оновлення.

> Якщо відео не з'являється — найчастіша причина зайва або відсутня кома
> між блоками. Перевірте розділові знаки.
