# ФІНАЛЬНІ КРОКИ — ключі вже вставлено

Firebase у вас повністю налаштовано, ключі вже вписані у файли.
Залишилось тільки залити їх на GitHub і підключити сповіщення на телефоні.

═══════════════════════════════════════
КРОК 1. ЗАЛИТИ ФАЙЛИ НА GITHUB
═══════════════════════════════════════

1. github.com → ваш репозиторій integrity-pwa.
2. «Add file» → «Upload files».
3. Розпакуйте архів і перетягніть ВСІ файли (перезапишуть старі):
   index.html, guide.html, sw.js, firebase-messaging-sw.js,
   manifest.webmanifest і теку icons.
4. «Commit changes». Зачекайте 1–2 хвилини.

═══════════════════════════════════════
КРОК 2. ПЕРЕВСТАНОВИТИ НА ТЕЛЕФОНІ
═══════════════════════════════════════

iPhone тримає стару версію в кеші, тому переустановлення обов'язкове:

5. Видаліть старий ярлик з головного екрана (утримати → Видалити).
6. Відкрийте https://uzknpu-sys.github.io/integrity-pwa/ у Safari.
7. «Поділитися» → «На початковий екран».
8. Відкрийте застосунок З ЯРЛИКА (не з Safari).

Тепер має бути:
   • офлайн-картка «Довідник стандартів доброчесності» (40 задач);
   • кнопка «🔔 Увімкнути сповіщення».

═══════════════════════════════════════
КРОК 3. УВІМКНУТИ СПОВІЩЕННЯ
═══════════════════════════════════════

9. Натисніть «🔔 Увімкнути сповіщення» → «Дозволити».
10. Має з'явитися «✓ Сповіщення увімкнено».

═══════════════════════════════════════
КРОК 4. НАДІСЛАТИ ТЕСТОВУ НОВИНУ
═══════════════════════════════════════

11. Firebase Console → Messaging → «Create your first campaign»
    → «Firebase Notification messages».
12. Заголовок + текст → далі → Target: ваш застосунок →
    Review → Publish.

✅ На телефоні з'явиться БАНЕР зверху. Це і є успіх.
   (Циферку на ярлику iPhone для PWA майже не показує — не чекайте її.)

═══════════════════════════════════════
ЯКЩО КНОПКА ВСЕ ОДНО НЕ З'ЯВИЛАСЯ
═══════════════════════════════════════

• Переконайтесь, що залили саме НОВІ файли (дата свіжа).
• Видаліть ярлик ще раз, у Safari закрийте всі вкладки сайту,
  зайдіть заново. iPhone інколи тримає кеш — повторне додавання скидає.
• Відкривайте лише з ярлика. У Safari кнопка теж видна, але
  сповіщення на iPhone працюють тільки із встановленого застосунку.
